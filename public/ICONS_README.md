# Icons and Images Needed

## Required Icons for SEO & App Stores

Please add the following icon files to the `/public` directory:

### Favicon
- **favicon.ico** (16x16, 32x32, 48x48 multi-size ICO file)

### Web App Icons
- **icon-192.png** (192x192 PNG) - For PWA and Android
- **icon-512.png** (512x512 PNG) - For PWA and Android

### Apple Touch Icons
- **apple-touch-icon.png** (180x180 PNG) - For iOS home screen

### Open Graph Image
- **og-image.png** (1200x630 PNG) - For social media sharing
  - Should feature the Rizzer logo and tagline
  - Optimized for Facebook, Twitter, LinkedIn previews

## Design Guidelines

All icons should:
- Use the Rizzer brand colors (pink #FE3C72, salmon #FF7854)
- Feature the Rizzer logo prominently
- Have transparent or white backgrounds
- Be optimized for web (compressed PNG)

## Tools to Generate Icons

You can use these tools to generate all required sizes:
- https://realfavicongenerator.net/
- https://www.favicon-generator.org/
- Figma or Adobe Illustrator for custom designs
